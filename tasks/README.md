## 🎯 How to Use This Index

### Starting a New Week:
1. Click the week link above (e.g., [Week 1](week1.md))
2. Follow the daily breakdown (Mon-Fri modules)
3. Complete Saturday hands-on labs
4. Update Excel tracker and Power BI dashboard on Sunday
5. Move to next week

### Daily Routine:
```
Morning (10 min):    Anki flashcard review
Evening (45 min):    Follow module from week file
                     Complete hands-on exercise
                     Update Excel tracker
Weekend (2-3 hours): Hands-on labs
Sunday (30 min):     Review, refresh Power BI, create flashcards
```

### Tracking Progress:
1. **Excel Tracker:** Open `my-dp700-progress.xlsx` and mark modules ✅
2. **Power BI Dashboard:** Refresh `DP700-ExamPrep.pbix` to see visual progress
3. **Flashcards:** Review daily in Anki app (aim for 10-15 min)
4. **This Index:** Update status (⬜ Not Started → 🟨 In Progress → ✅ Complete)
