## 📝 Exam Strategy

### Question Types You'll See:

1. **Knowledge-based:** Recall facts, definitions, concepts
2. **Scenario-based:** Apply knowledge to realistic situations
3. **Decision-based:** Choose best tool/approach for scenario
4. **Troubleshooting:** Identify issue and solution
5. **Best practice:** Select recommended approach

### For Each Question:

```
1. Read the full question carefully
2. Identify what's being asked
3. Eliminate obviously wrong answers
4. Consider all remaining options
5. Choose best answer
6. If unsure, flag and move on
7. Return to flagged questions if time permits
```
