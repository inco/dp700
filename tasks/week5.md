## 🔬 Hands-On Labs

### Lab 5.1: Dimensional Model Design (90 min)
- [ ] Design star schema for retail scenario
- [ ] Create dimension tables (Customer, Product, Date, Store)
- [ ] Create fact table (SalesOrder)
- [ ] Implement surrogate keys
- [ ] Document schema design

### Lab 5.2: Data Loading Patterns (45 min)
- [ ] Full load for dimensions
- [ ] Incremental load for facts
- [ ] Implement SCD Type 2
- [ ] Test MERGE statements

### Lab 5.3: Query Optimization (45 min)
- [ ] Create baseline queries
- [ ] Add clustered columnstore indexes
- [ ] Implement partitioning
- [ ] Compare before/after performance

### Lab 5.4: Security Implementation (30 min)
- [ ] Row-level security
- [ ] Column-level security
- [ ] Dynamic data masking
- [ ] Test access controls
